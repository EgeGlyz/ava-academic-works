# Currency Converter (Java)

A robust and efficient desktop application built with Java to perform real-time currency conversions. This project focuses on clean code, object-oriented principles, and accurate mathematical logic.

## 🚀 Features
* **Multi-Currency Support:** Convert between various global currencies (USD, EUR, TRY, etc.).
* **Precision Logic:** Handles decimal calculations with high accuracy.
* **Input Validation:** Built-in error handling to prevent invalid characters or empty inputs from crashing the app.
* **Scalable Architecture:** Designed using Object-Oriented Programming (OOP) for easy integration of live APIs in the future.

## 🛠 Tech Stack
* **Language:** Java
* **Environment:** IntelliJ IDEA
* **Version Control:** Git / GitHub

## 📂 Project Structure
* `src/`: Contains the source code files.
* `out/`: Compiled bytecode (not tracked in Git).

## 📝 How to Run
1.  Clone the repository:
    ```bash
    git clone [https://github.com/ege-gulyaz/currency-converter.git](https://github.com/ege-gulyaz/currency-converter.git)
    ```
2.  Open the project in **IntelliJ IDEA** or any Java IDE.
3.  Run the `Main.java` file.
